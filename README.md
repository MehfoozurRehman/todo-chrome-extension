# Todo Chrome Extension

A simple and efficient Todo List Chrome Extension built with React and Vite. Easily manage your daily tasks directly from your browser.

## Features

- Add, edit, and delete tasks
- Persistent storage using browser local storage
- Clean and minimal UI
- Fast and lightweight

## Tech Stack

- React
- Vite

## Getting Started

### Development

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
```
